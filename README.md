# KeyManager

Приложение для управления ключами (WPF + PostgreSQL)

## Стек
- C# (WPF)
- Postgresql

## Требуемый функционал
- Управление организациями (CRUD)
- Управление ключами (CRUD, блокировка)
- Фильтрация ключей по организации и поиск по значению
- Экспорт списка ключей в Word (дополнительно)

🚀 Инструкция по запуску проекта
Для корректной работы приложения убедитесь, что в вашей системе установлены .NET 10 SDK и СУБД PostgreSQL.

1. Подготовка репозитория
Склонируйте проект на локальный компьютер:

```Bash
git clone <ссылка_на_репозиторий>
cd <название_папки_проекта>
```
2. Настройка базы данных
Перед запуском необходимо указать строку подключения к вашей базе данных PostgreSQL:

Перейдите в директорию Initialization.

Откройте файл appsettings.json.

В секции ConnectionStrings измените значение DefaultConnection:

```JSON
"ConnectionStrings": {
  "DefaultConnection": "Host=localhost; Port=5432; Database=YourDbName; Username=your_user; Password=your_password"
}
```
3. Сборка и миграции
Откройте терминал в корневой папке проекта и выполните следующие команды:

Сборка проекта:
```Bash
dotnet build
```

Применение миграций:
```Bash
dotnet run -- migrate
```

4. Запуск приложения
После успешного выполнения миграций вы можете запустить проект:

Через терминал: `dotnet run`

Или запустив скомпилированный файл .exe в директории bin/Debug/net10.0/.
