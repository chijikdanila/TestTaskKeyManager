# KeyManager

Приложение для управления ключами (WPF + PostgreSQL)

## Стек
- C# (WPF)
- Postgresql

## Требуемый функционал
- Управление организациями (CRUD)
- Управление ключами (CRUD, блокировка)
- Фильтрация ключей по организации и поиск по значению
- Экспорт списка ключей в Word (дополнительно)
